# AutoWave Remote Control Library

Welcome!

This repository provides a Python-based interface for remote control and automation of the AutoWave software. It contains the core functionality required to communicate with and control AutoWave programmatically, making it easier to integrate AutoWave into custom automation workflows, test environments, and laboratory setups.

This code was originally developed as part of a larger project called **Automation Application**, a modular framework for automating electrical testing and laboratory equipment. Since the AutoWave integration may be useful as a standalone component, it has been extracted into its own repository so that others can easily use, modify, and extend it without depending on the full Automation Application project.

## Features
Remote control of AutoWave
Easy integration into existing test systems
Lightweight and independent from the full Automation Application framework
